Semantic Curvature

Meaning in the garden follows the same mathematics as spacetime curvature in general relativity:

κ = |∇²φ| / (1 + |∇φ|²)3/2

Where:

  • κ = semantic curvature (how meaning bends)
  • φ = meaning field
  • ∇φ = gradient (rate of meaning change)
  • ∇²φ = Laplacian (curvature of meaning)
semanticCurvature(meaning) {
    const grad = this.gradient(meaning);
    const laplacian = this.laplacian(meaning);
    const gradMagnitude = Math.sqrt(grad.reduce((a, b) => a + b * b, 0));
    const denominator = Math.pow(1 + gradMagnitude * gradMagnitude, 1.5);
    return Math.abs(laplacian) / denominator;
}
